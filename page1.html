    1. HTML
      1. HTML is a markup language used to describe the structure, not the style, of websites.
      2. Elements in an HTML document are arranged into a logical tree structure called the DOM.
    2. CSS
      1. CSS uses the box model, which assigns every element some amount of padding, border, and margin spacing.
      2. CSS provides multiple methods for structuring page layouts. For this class, we're using grid-template-areas.
    3. CSS Colors
      1. CSS predefines 140 colors which can be used by name.
      2. Other colors can be assigned to elements using RGB or hexadecimal values.
      3. RGB and hex values define a color based on its amounts of red, green, and blue; each color can be assigned an intensity from 0 - 255.
